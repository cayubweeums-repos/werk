# werk
werk is a workout app that I always wish was out there

# prereqs

Install poetry and python

# Setup

```
git clone # this repo
cd werk
poetry install
```

# Setup local dev env
```
# For conventional commits and versioning
sudo apt-get install npm -y

npm install -g commitizen
```

For all commits
```
git cz
```

# Running
```
poetry run python main.py
```
